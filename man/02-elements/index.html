ЕЛЕМЕНТИ

СИНОПСИС

Контрольні елменти — це компоненти рівня веб-фреймворку NITRO. Тут представлені усі спеціалізовані контролні елменти які специфічні для продукту МІА:Документообіг, та не входять в поставку NITRO. Серед них зокрема основні панелі продукту, контрольний елемент управління ділопроцесами документообігу, контрольні елементи для пошуку, редагування та створення документів, елементи для завантаження файлів, тощо.\

Контрольні елементи

BPE.Control

Контрольний елемент управління ділопроцесом.

BPE.Monitor

Контрольний елемент відображення ділопроцесів РМК картки.

CRM.Search

Контрольний елемент пошуку документів.

CRM.FileUpload

Контрольний елемент завантаження файлів.

NITRO.Table

Контрольний елемент відображення таблиці документів.

CRM.Control

Події сторінки зі списком документів (CRM.HTM).

RMK.Control

Події сторінки реєстраційно-моніторингової картки (RMK.HTM).
